=head2 NAME

 icewm-set-gnomewm - modify GNOME settings to use IceWM

=head2 SYNOPSIS

B<icewm-set-gnomewm> [I<OPTIONS>]

=head2 DESCRIPTION

B<icewm-set-gnomewm> is a helper script which instruments GNOME to
replace its own WM with icewm.

=head2 OPTIONS

=head2 GENERAL OPTIONS

=over

=item B<-h>, B<--help>

Print a brief usage statement to F<stdout> and exit.

=item B<-V>, B<--version>

Print the program version to F<stdout> and exit.

=back
